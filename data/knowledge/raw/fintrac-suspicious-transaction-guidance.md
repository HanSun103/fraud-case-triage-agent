# FINTRAC suspicious transaction guidance placeholder

This local-first demo file is a placeholder for manually added authoritative guidance.

Analysts should review unusual cross-border activity, abrupt increases in transaction amount relative to normal customer behavior, rapid transaction velocity, new-device activity, and unexplained first-time merchant usage.

Location mismatch on its own may be weak evidence when customer travel is already known. It should be weighed together with device familiarity, merchant history, prior spending pattern, and any analyst notes.

Case notes should record why a transaction appears suspicious, which signals raised concern, what context reduced concern, and why escalation or closure was recommended.
